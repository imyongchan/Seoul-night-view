# Seoul Night View (야경맛집)

서울의 아름다운 야경 명소를 소개하는 웹 애플리케이션입니다. 서울시 공공데이터 API를 활용하여 야경 명소 정보를 제공하며, 검색, 지도 보기, 추천 기능으로 사용자에게 편리한 야경 탐험 경험을 선사합니다.

## 프로젝트 소개

이 프로젝트는 혼자서 일주일 동안 개발한 간단한 웹 애플리케이션으로, 서울의 야경 명소를 쉽게 찾고 탐험할 수 있도록 도와줍니다. 서울시에서 제공하는 공공데이터 API를 적극적으로 활용하여 신뢰할 수 있는 정보를 기반으로 하며, 직관적인 UI/UX로 누구나 쉽게 사용할 수 있습니다.

프로젝트의 주요 목표는 다음과 같습니다:
- 서울의 숨겨진 야경 명소를 사용자들에게 소개
- 공공데이터 API를 활용한 실질적인 웹 개발 경험 축적
- 반응형 웹 디자인 구현
- 사용자 친화적인 인터페이스 제공

## 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업과 구조화된 웹 페이지 구성
- **CSS3**: 반응형 디자인, 모던한 UI 컴포넌트 구현
- **JavaScript (ES6+)**: 동적 기능 구현 및 API 연동
- **jQuery**: DOM 조작 및 이벤트 처리 간소화

### APIs & Libraries
- **서울시 공공데이터 API**: 야경 명소 데이터 제공
- **Kakao Maps API**: 지도 표시 및 위치 기반 서비스
- **Font Awesome**: 아이콘 라이브러리

### Development Tools
- **Visual Studio Code**: 개발 환경
- **Git**: 버전 관리

## 주요 기능

### 1. 야경 명소 검색
- **키워드 검색**: 명소 이름으로 빠른 검색
- **지역 필터링**: 용산구, 중구, 마포구 등 구별 필터링
- **테마 필터링**: 로맨틱, 도시경관, 자연경관 등 테마별 검색
- **가격대 필터링**: 무료/유료 구분 및 가격대별 필터링
- 실시간 검색 결과 표시 (총 51개 야경 명소 데이터)

### 2. 야경 명소 지도
- **Kakao Maps 연동**: 서울시 야경 명소 위치 표시
- **마커 클러스터링**: 효율적인 지도 정보 표시
- **인포윈도우**: 명소 상세 정보 팝업 (주소, 교통편, 주차 정보)
- **내 위치 찾기**: 현재 위치 기반 주변 야경 명소 탐색

### 3. 추천하기
- **사용자 추천 시스템**: 숨겨진 야경 명소 공유
- **폼 검증**: 입력 데이터 유효성 검사
- **이메일 형식 검증**: 올바른 이메일 주소 입력 확인

### 4. 반응형 디자인
- **모바일 최적화**: 스마트폰, 태블릿 지원
- **햄버거 메뉴**: 모바일 환경에서의 네비게이션
- **플렉시블 레이아웃**: 다양한 화면 크기 대응

- **클라이언트 사이드 렌더링**: JavaScript를 통한 동적 콘텐츠 생성

## 실행 방법

### 1. 프로젝트 다운로드
```bash
git clone https://github.com/imyongchan/Seoul-night-view.git
cd Seoul-night-view
```

### 2. 로컬 서버 실행
웹 브라우저에서 직접 HTML 파일을 열어도 실행 가능하지만, 로컬 서버를 사용하는 것을 권장합니다.

#### Python을 사용하는 경우:
```bash
# Python 3.x
python -m http.server 8000

# 또는 Python 2.x
python -m SimpleHTTPServer 8000
```

#### Node.js를 사용하는 경우:
```bash
npx http-server -p 8000
```

#### VS Code Live Server 확장 사용:
1. VS Code에서 프로젝트 열기
2. Live Server 확장 설치
3. HTML 파일에서 우클릭 → "Open with Live Server"

### 3. 브라우저에서 접속
```
http://localhost:8000/html/index.html
```

### 4. 기능 테스트
- 메인 페이지에서 각 메뉴 탐색
- 검색 페이지에서 필터 적용 및 검색
- 지도 페이지에서 위치 정보 확인
- 추천 페이지에서 폼 제출 테스트
